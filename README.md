# monocopy

Static copy-safe text page for factory embroidery monograms.

Part of the L134 Monogram Copy QR sprint. Consumes base64url-encoded JSON payloads via `?t=` query param and renders them as clipboard-friendly cards.

Deployed via Cloudflare Pages at https://monocopy.pages.dev/.
